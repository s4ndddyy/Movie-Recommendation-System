# Movie Recommendation System
An item based collaborative filtering system using the MovieLens 100K dataset. It recommends movies based on cosine similarity of user ratings and allows browsing by movie name or genre.

## Features
- Search movies by name and get top recommendations.  
- Browse movies by genre.  
- Interactive web interface using Streamlit.

## Dependencies
- Python
- Pandas
- Scikit learn
- Streamlit

## Datasets
  - MovieLens100k (included in the files)
